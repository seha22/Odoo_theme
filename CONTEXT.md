# CONTEXT - Odoo_theme

## Project Overview
Custom Full Theme Module untuk Odoo 19 yang ditujukan untuk online shop penjualan Sarung Jok Mobil.

## Key Decisions

- **Odoo Version**: 19
- **Theme Type**: Full Theme Module (menggantikan sebagian besar tampilan default)
- **Focus**: Professional & Modern UI/UX dengan motion effect dan parallax pada Homepage
- **Business Domain**: E-commerce untuk produk Sarung Jok Mobil (car seat covers)

## Terminology
- Full Theme: Tema yang berdiri sendiri dan mengoverride template serta aset default Odoo Website + Website Sale.
